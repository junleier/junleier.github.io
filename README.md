# 旧信里的香樟味
** 
梅雨季节的江南老巷，青石板路被雨水浸得发亮。林晚推着吱呀作响的自行车，车筐里躺着刚收的旧书，拐进巷尾那家 “晚来书坊” 时，雨丝正斜斜掠过窗棂。她弯腰擦拭书架上的灰尘，指尖忽然触到一本烫金封面的《雪莱诗选》，书页间滑落一张泛黄的信笺。

字迹清隽，带着少年人特有的挺拔：“致未来的读者：若你遇见这封信，可否替我看看巷口那棵香樟树？它去年结了很多紫黑色的小果，落在地上会染紫鞋底。” 落款是 “陆知年”，日期是七年前的梅雨日。

林晚的心轻轻一颤。她来这座小城开书店三年，每日都要经过巷口那棵枝繁叶茂的香樟，却从未留意过它的果实。次日清晨雨停，她特意驻足树下，阳光穿过层层叠叠的叶片，在地上投下斑驳的光影。风一吹，真有几颗紫黑色的小果滚落，沾在她的白帆布鞋上，晕开浅浅的紫痕。

她把信笺夹回书中，心里藏了个小小的秘密。往后的日子，她总会在整理旧书时格外留意 “陆知年” 这个名字，却再也没找到相关的只言片语。直到三个月后，一个穿着白衬衫的青年推开书店的门，目光扫过书架时，忽然停在《雪莱诗选》上。

“这本书……” 他声音温润，带着一丝不易察觉的激动。

林晚抬头，撞进一双清澈的眼眸，像雨后初晴的天空。“你在找它？” 她取下那本书，信笺依旧夹在原来的页码间。

青年指尖抚过泛黄的纸页，眼底泛起笑意：“七年前我来这里出差，在书店避雨时写下这封信，没想到真的有人看到了。” 他自我介绍是建筑设计师，七年前因项目短暂停留，如今是来参加老城区改造的研讨会。

“我叫林晚。” 她递过一杯温热的桂花茶，“我替你看过香樟树了，它每年都会结很多小果。”

陆知年的眼睛亮了起来。接下来的日子，他成了书坊的常客。有时两人并肩整理旧书，指尖偶尔相触，便会像受惊的小鹿般分开；有时在窗边对坐，听雨声敲打窗棂，分享彼此喜欢的诗句。林晚发现，陆知年不仅懂建筑，还对文学有着浓厚的兴趣，他总能精准地说出她喜欢的作家的冷门作品。

一次闲聊中，林晚说起自己开书店的初衷：“小时候外婆家有个旧书箱，我总在里面翻找童话书，那种油墨香让我觉得很安心。” 陆知年沉默片刻，从背包里拿出一个磨损的笔记本，里面夹着一张老照片 —— 一个小女孩蹲在书箱前，笑得眉眼弯弯。

“这是……” 林晚惊讶地捂住嘴。
“七年前我在巷口拍照，恰好拍到了你。” 陆知年的脸颊泛起微红，“其实我一直记得你，只是当时没敢上前打招呼。这次回来，除了工作，也是想找找那个蹲在书箱前的女孩。”
原来所有的相遇都不是偶然。梅雨再次落下时，陆知年牵着林晚的手走在香樟树下，紫黑色的小果落在两人的肩头。他从口袋里掏出一张新的信笺，写下：“致林晚：与你相遇，是时光最温柔的馈赠。”
林晚接过信笺，鼻尖萦绕着淡淡的香樟味和油墨香。她知道，这段始于旧信的缘分，终将在岁月里沉淀成最绵长的温柔，就像巷口的香樟树，年复一年，枝繁叶茂。
